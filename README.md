# CRUD Application Documentation

## Introduction
This repository contains a CRUD (Create, Read, Update, Delete) application designed to manage client records. The application provides a user-friendly interface and allows users to easily interact with client data.

## Technologies Used
- **Front-end:** HTML, CSS, JavaScript
- **Back-end:** Node.js, Express.js
- **Database:** MongoDB
- **Version Control:** Git
- **Deployment:** Heroku (or any other platform used)

## Features
- Add new client records
- View existing client records
- Update client information
- Delete client records
- Search and filter options for better usability

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/GustavoTI-DEv/crud-clientes.git
   ```

2. Navigate to the project directory:
   ```bash
   cd crud-clientes
   ```

3. Install the necessary dependencies:
   ```bash
   npm install
   ```

4. Set up the database (if applicable).

5. Run the application:
   ```bash
   npm start
   ```

6. Access the app in your browser at `http://localhost:3000`.
